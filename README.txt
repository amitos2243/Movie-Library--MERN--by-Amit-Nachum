 • Server Folder - Which includes all the BackEnd files, written in Node.js & originally connected to a Mongo Database created by me, tested with Postman CLIENT.

 • Client Folder - Which includes all the FrontEnd files - written in the REACT Framework - implementing Javascript , HTML & Styled using MUI (Material UI) & CSS.

 • a Schema file for the project, created by me, using Diagrams.net. ( formerly known as draw.io)
